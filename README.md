# Extra Hashes

This plugin uses HashLibPlus to add various new hash functions to the Hashes view.
Currently supported functions are:
- Adler32
- AP
- -BKDR
- Bernstein
- Bernstein1
- DEK
- DJB
- ELF
- FNV1a_32
- FNV32
- HS
- OneAtTime
- PJW
- Rotating
- RS
- SDBM
- ShiftAndXor
- SuperFast
- Murmur2_32
- MurmurHash2_x86_32
- XXHash32
- Jenkins3
- FNV64
- FNV1a_64
- Murmur2_64
- SpiHash64_2_4
- XXHash64
- SpiHash128_2_4
- MurmurHash3_x86_128
- MurmurHash3_x64_128
- Tiger
- Tiger2
- Blake2B
- Blake2S
